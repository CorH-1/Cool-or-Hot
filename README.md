## local 개발 환경 setting

1. 가상환경 설정

```
# 가상환경 생성 및 접속
$ python -m venv venv
$ source . venv/bin/activate

# 설치항목
$ pip install -r requirements.txt
```

2. 브랜치 설정

```
# main 브랜치에서 분깃
$ git checkout -b develop
$ git checkout -b {생성할 브랜치 이름} develop

# 브랜치 구분
main : 배포용 [merge 금지]
develop : 개발용
hotfix : 버그 혹은 오류 수정용
```
